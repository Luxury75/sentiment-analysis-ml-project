# Sentiment Analysis ML Project

> NLP sentiment classifier achieving 84.5% accuracy on customer reviews

## 🎯 Overview

Built and compared three machine learning models for sentiment classification:
- **SVM**: 84.5% accuracy ⭐ Best performer
- **Logistic Regression**: 79.4% accuracy
- **Naive Bayes**: 78.8% accuracy

## 🛠️ Tech Stack

- **Python**: scikit-learn, nltk, pandas, matplotlib
- **NLP**: Text preprocessing, TF-IDF, lemmatization
- **Models**: SVM, Naive Bayes, Logistic Regression
- **Evaluation**: Confusion matrices, classification reports

## 📊 Key Results

- Sentiment classification: Positive/Neutral/Negative
- Star rating prediction: 1-5 stars (0.63 MAE)
- Outperformed VADER baseline by 5%

## 🔍 Features

- Complete text preprocessing pipeline
- Multiple ML model comparison
- VADER baseline benchmarking
- Live prediction function
- Comprehensive visualizations
